Retrieve location info using Google maps API

Demo: Enter your zip/postal code












Documentation

Example usage

<script src="https://maps.google.com/maps/api/js?sensor=false"></script>
<script src="get-location.min.js"></script>
<script>
getLocation({
	id: 	'zip',
	city:	'city',
	state:	'state',
	country: 'country'
});
</script>

getLocation() parameters

Parameter NameDefault ValueDescription
id (string) (required) ID of Zip/Postal code field to be filled in
on (string) keyup (optional) field listener on when to trigger function. Possible values: onclick, onchange, onkeyup, onkeydown
replace (boolean) true (optional) change value of city, state and country fields using the supplied IDs.
city (string) (required) ID of the city field to be automatically populated
state (string) (required) ID of the state field to be automatically populated
country (string) (required) ID of the country field to be automatically populated
